# {{cookiecutter.extension_name}} JupyterLab extension

A JupyterLab extension for rendering {{cookiecutter.mime_short_name}}

## Prerequisites

* `jupyterlab>=0.11`

## Development

Install dependencies and build Javascript:

```bash
npm install
```

Re-build Javascript:

```bash
npm run build
```

Watch `/src` directory and re-build on changes:

```bash
npm run watch
```
